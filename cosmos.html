一切的开始
宇宙最初的状态，物理学家叫它奇点——意思是：在那一刻，所有的数学公式都算不出结果，所有已知的物理规律都失效了。我们能描述的只有"它之后"发生了什么，"它本身"是什么，目前没有人知道。

那时候温度高到大约 10³² 度——比太阳核心还要热一亿亿亿倍。热到任何你知道的东西都不可能存在。所有的能量被挤在一个极小极小的范围里。

然后——没有人知道为什么——它开始膨胀。

"大爆炸"这个名字其实起错了。它不是一个炸弹在某个地方炸开。它是空间本身在变大——每一个点和每一个点之间的距离都在变大。像一壶滚烫的水倒进一个越来越大的池子里，水还是那些水，但温度降了，因为摊薄了。

温度一降，一件事就发生了：粒子开始能够稳定存在。之前温度太高，就算粒子短暂出现了，也会立刻被撞碎。现在温度够低了，有些粒子终于能活下来不被摧毁了。
什么是"时空"？
时间和空间不是两样分开的东西，它们是一张布的两面。你在空间里移动得越快，你的时间就走得越慢——这不是比喻，是爱因斯坦用实验验证过的事实。

什么是"场"？
想象一片无边无际的海。海面上局部鼓起来的一个浪包 = 粒子。所以"场"是那片海本身。粒子不是独立的小球，它是场的一部分局部震动起来了。

什么是"能量"？
能量是"让事情发生的能力"。热、光、运动——只是能量的不同表现形式，可以互相转换，但总量不变。
原始状态 — 所有能量挤在极小范围，温度高到粒子无法存在
宇宙大爆炸 — 不是炸弹，是空间本身在变大
为什么宇宙不为零 — "什么都没有"反而是最不自洽的状态
t < 3 分钟
粒子时代
温度继续降。夸克第一次能稳定地组合在一起了。三个夸克被强力绑在一起，形成质子或中子。

质子带正电，中子不带电。质子的数量决定了一个原子是什么元素：1个 = 氢，6个 = 碳，26个 = 铁，92个 = 铀。质子数 = 元素的身份证号。

还有电子——比质子轻得多，带负电，到处散着。光子——光的最小单位。中微子——极其轻，能直接穿过整个地球。

宇宙诞生 3 分钟时，质子和中子开始组合——氢核、氦核、少量锂核。宇宙变成了一锅"核汤"。
四种力：
① 强力 — 把夸克绑成质子中子。力最强，只管原子核内部。
② 电磁力 — 控制电荷、光、化学反应。日常几乎所有现象都是它。
③ 弱力 — 让粒子变身，核聚变和放射性衰变需要它。
④ 引力 — 质量让时空弯曲。力最弱但管最远。

质量 = 一个东西有多难推动。越大越难推，也越能让时空弯曲。
夸克 — 最基本的砖头，三个绑一起 = 质子或中子
质子 — 带正电，质子数 = 元素身份证号
中子 — 不带电，帮质子们待在一起
电子·光子·中微子 — 到处飞的三种轻粒子
核汤 — 原子核 + 电子 + 光子散着飘，还没组装成原子
~38 万年
宇宙变透明
38 万年里，宇宙一直是一片"白雾"——光子不停地撞电子，走不了直线。

终于冷到约 3000 度。光子不够力气把电子撞飞了。电子被原子核吸住——第一批完整的原子诞生。光子突然可以自由飞行。宇宙从"白雾"变成透明。

那一刻释放出来的光子今天还在飞。波长被拉长成了微波——宇宙微波背景辐射，宇宙婴儿时期的"照片"。
星系转得太快——能看见的物质的引力不够。说明有看不见的东西在提供额外引力。它占宇宙总质量的 85%。

它先聚成"引力坑"，普通物质掉进去才形成恒星和星系。宇宙先长骨架（暗物质），再长肉（普通物质）。
宇宙变透明 — 电子被接住 → 光子自由 → 白雾消散
宇宙微波背景辐射 — 38万年前的光，今天还在飞
暗物质 — 看不见但有引力，宇宙结构的骨架
~1 亿年
第一代恒星 — 元素工厂
黑暗时期。没有恒星，没有光源，到处是氢和氦。

但宇宙不均匀。密一点的地方引力强一点，拉来更多气体——正反馈。当温度到约一千万度——核聚变点燃。第一代恒星诞生。

它们极端巨大，几十到上百个太阳质量，只活几百万年。核心是元素锻造厂：氢→氦→碳→氧→硅→铁。铁是终点——之后聚变要吸收能量。

铁堆积，核心塌缩，不到一秒——反弹——超新星爆炸。瞬间造出铁到铀的所有元素，炸进宇宙。

你身体里的碳、血液里的铁、骨头里的钙——全是某颗恒星爆炸时喷出来的。
核聚变 — 轻核合成重核，少掉的质量变能量
第一代恒星 — 纯氢氦，超大质量，内部锻造氢到铁
超新星爆炸 — 铁核塌缩 → 反弹 → 炸出 26-92 号元素
~2 亿 – 10 亿年
第二代恒星 → 星系成型
爆炸后宇宙有了重元素。重元素帮气体散热——相当于装了散热器。气体能降温、碎裂成小块，形成大量大小不一的恒星。

大量恒星 + 暗物质骨架 + 气体 = 星系。暗物质做骨架，气体做肌肉，恒星做灯光，黑洞做心脏。
小恒星（<8太阳质量）→ 外层吹掉 → 核心变白矮星
大恒星（8-25太阳质量）→ 超新星 → 核心变中子星
极大恒星（>25太阳质量）→ 核心变黑洞
第二代恒星 — 有重元素 → 能散热 → 大量小恒星形成
星系 — 暗物质骨架 + 恒星 + 气体 + 黑洞心脏
~92 亿年
太阳系诞生
某颗恒星爆炸，冲击波挤压附近的气体和尘埃，触发坍缩。角动量守恒使它们转成扁平圆盘。中心温度达到核聚变——太阳点燃。

距太阳近的地方太热，只有岩石和金属能存在——形成水星、金星、地球、火星。远处冷，冰和气体能留下——形成木星、土星等气体巨行星。
太阳 — 气体坍缩 → 旋转成盘 → 核心点燃
冰线 — 分割了岩石行星和气体行星的边界
~92 亿 – 96 亿年
地球的诞生与演化
早期太阳系像台球桌——到处碰撞。一个火星大小的天体Theia撞上原始地球，整颗星熔化成岩浆球。碎片聚集成月球。

之后铁等重金属下沉到核心，释放的势能让地球更热。形成分层结构：铁核 → 地幔 → 地壳。

地幔极热产生对流——像锅里煮汤。这个对流后来驱动了板块运动。

火山喷发释放水汽、二氧化碳——形成原始无氧大气。温度终于降到 100°C 以下，千年暴雨落下，积成了最早的海洋。

闪电、紫外线、液态水——化学汤正在沸腾。生命即将破壳。
Theia 碰撞 — 全星熔化，碎片变月球
铁下沉 — 重的沉底，势能变热，形成地核
对流 — 热的上升冷的下沉，驱动板块运动
放气 — 岩石里的气体被烤出来 → 原始大气
海洋 — 水蒸气反复蒸发-凝结，最终积成海
舞台就绪 — 闪电 + 紫外线给化学汤注入能量
