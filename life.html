~38 亿年前
生命的起源（原核细胞）
生命 = 一个能够自我复制的、有边界的、化学系统。不多不少。 为什么"自我复制"关键？能复制的积累，不能复制的消失。复制出错（突变）→碰巧复制更快→扩散。自然选择在"生命"这个词出现之前就在运行了。 环境永远都是准备好了的。任何环境都必然会产生某些结果。地球这个环境产生的结果就是生命。
因果链：无机分子 → 能量驱动 → 有机分子 → RNA自我复制 → 磷脂膜包裹 → 第一个"细胞" → 囊泡竞争 → DNA存储 → 蛋白质执行 → 原核生物
分子 · 离子 · 氧化 · 有机分子
▾
分子=原子用化学键绑一起。离子=丢/抢电子后带电的原子。氧化=被抢走电子。 有机分子=以碳为骨架的分子。为什么碳？外层恰好4个电子→既不全丢也不全抢→选择共享→能同时朝四个方向连接→唯一能搭复杂三维结构的元素。 原始海洋中：CO₂+H₂O+N₂+紫外线/闪电 → 有机分子。米勒-尤素实验已在实验室重现——纯化学，不需要魔法。
原始核糖核酸（RNA）· 核苷酸
▾
核苷酸=磷酸+糖（核糖）+碱基。RNA的基本单元。 RNA=核苷酸串成的长链，四种碱基（A/U/C/G）顺序可变→携带信息。 RNA有特殊性质：能催化化学反应。如果一段RNA碰巧能催化核苷酸连接→就能加速造自己的副本→自我强化循环→第一个自我复制系统。 但还不是生命——缺边界。为什么？没边界→有用分子扩散到整个海洋→浓度太低→反应速率趋近零。有边界→分子被圈在小空间→浓度高→反应高效。
磷脂 · 脂肪泡泡 · 选择通透性
▾
磷脂：一头亲水一头怕水→扔水里自动排成双层膜→中空小球=囊泡。纯物理化学自发行为。 选择通透性：膜不是铁壁→小分子能穿过，大分子被挡→有了"内"和"外"→系统有了边界→可以积累有用分子、维持自己的化学环境。
囊泡竞争 · 自我复制
▾
磷脂囊泡+里面的RNA→最原始"细胞"。有限资源下复制快的赢，复制慢的被淘汰。
氨基酸 · 蛋白质 · 蛋白质合成机制
▾
氨基酸：20种小分子。蛋白质：氨基酸按特定顺序串成链→折叠成特定形状→形状决定功能。蛋白质负责执行：搭结构（支架）、催化反应（酶）、运输、传信号。 合成流程：DNA→转录成mRNA→mRNA指导氨基酸排列→蛋白质折叠。
脱氧核糖核酸（DNA）· ATP · 葡萄糖 · 脂肪分子
▾
DNA双链比RNA稳定→长期信息存储。为什么更稳定？①双链=两份备份，一条坏了照另一条修 ②糖骨架少一个氧原子→化学上更难被分解。 蛋白质为什么比RNA干活强？RNA只有4种碱基当积木，蛋白质有20种氨基酸→折叠出的形状多几个数量级→能做的事多几个数量级。 DNA=图纸档案馆，RNA=施工副本，蛋白质=按图纸建的结构和工人。 ATP：细胞"充电电池"。葡萄糖：基础能量燃料。脂肪：长期储能，每克比糖多一倍。
原核生物——第一批真正的生命
▾
结构：没有细胞核（DNA直接漂在细胞质里）、没有内部隔间。一层膜包着：环形DNA、核糖体（读RNA造蛋白质的小机器）、细胞质。极简但完整——能复制、有边界、有代谢。 它们统治地球将近20亿年。地球大部分时间只有细菌。
生命是什么——一个更深的视角
▾
~35–25 亿年前
早期生命的演化：光合作用和大氧化事件
最早的细菌靠地热、化学物质获取能量——但这些有限→生命总量有天花板。直到一个细菌找到了新能量源。
因果链：无氧发酵（低效）→ 光合作用（无限能量）→ O₂"废料" → 7亿年被还原性物质吸收 → 耗尽 → O₂暴涨 → 大氧化 → 厌氧生命大灭绝 → 有氧呼吸（能量×18）→ 臭氧层 → 为登陆铺路
"无氧发酵" → 光合作用
▾
无氧发酵：没氧气时分解有机物获取少量ATP。效率低——葡萄糖没被完全拆解，大部分能量还锁着。 光合作用：6CO₂+6H₂O+光能→葡萄糖+6O₂。太阳能几乎无限→只需阳光水CO₂就行。 某个细菌蛋白质碰巧捕捉光子→微小优势→繁殖稍快→几百万年优化→完整光合系统。氧气是"废料"——从H₂O提取电子后剩下的就是O₂。
还原性物质 · 为什么O₂花了7亿年才积累
▾
地球有大量还原性物质（Fe²⁺、H₂S等）→O₂一释放就被"吃掉"：Fe²⁺+O₂→Fe₂O₃（铁锈）。缓冲期7亿年。直到还原性物质耗尽→O₂终于积累。
大氧化事件 · 活性氧 · 超氧化物歧化酶
▾
~25亿年前O₂从~0%暴涨到20-30%=大氧化事件。O₂对厌氧生命剧毒——产生活性氧（ROS），攻击DNA、膜、蛋白质。大灭绝。 活下来的两条路：①躲到无氧环境 ②进化出超氧化物歧化酶→把毒药变燃料→有氧呼吸。
细胞呼吸 · 臭氧层
▾
葡萄糖+O₂→CO₂+H₂O+大量ATP。效率比无氧发酵高~18倍——O₂能把葡萄糖彻底拆解。能量×18→复杂生命的能量基础。 O₂在高空→臭氧层→挡紫外线→之前生命只能躲水下→现在陆地变得宜居。
~20 亿年前
真核生物的出现和内共生理论
原核生物的限制：没有内部分工→所有反应在同一空间→互相干扰→复杂度有天花板。 真核生物的革命：细胞核（DNA被保护）+各种细胞器各司其职→一个人干所有活 变成 有分工的工厂。
内共生——为什么没消化掉？
▾
大原核细胞吞了小原核细胞→但没消化。为什么？小细胞擅长有氧呼吸→不断产ATP→留着它比吃掉它更赚。久了之后基因互相依赖→谁也离不开谁→小的变成"内部器官"。
线粒体 · 叶绿体
▾
线粒体=被吞的有氧呼吸细菌→"发电厂"。证据：有独立的环形DNA（跟细菌一样）、双层膜、二分裂。 叶绿体=被吞的蓝藻→"太阳能板"。同样有独立DNA。 线粒体+叶绿体→真核细胞能量远超原核→复杂生命从此可能。
细胞核怎么来的 · "无聊的十亿年"
▾
细胞核：细胞膜向内折叠→把DNA包起来→转录和翻译被隔开→加了"质检"环节→错误率降低、调控更精细。 真核（~20亿年前）到多细胞（~6亿年前）之间等了14亿年。为什么？多细胞需要四套全新基础设施：细胞黏附蛋白（粘住不散架）、细胞间信号通讯、程序性细胞死亡（某些细胞主动自杀）、基因调控开关（同一DNA在不同细胞读不同段落）。每套都需要几亿年进化。
~6–5.4 亿年前
多细胞生物的进化和寒武纪大爆发
单细胞能做的有限。细胞合作→分工→效率暴涨。前提：①足够能量（线粒体解决）②细胞间通讯和黏附机制（需要时间进化）。
从细胞到器官 · 组织 · 肌肉细胞
▾
多细胞的核心：单个细胞放弃全能性，换取集体的超能力。同类细胞聚成组织（肌肉组织、上皮组织、神经组织）→ 不同组织协作成器官（心脏=肌肉+结缔+神经）→ 器官配合成系统（循环系统、消化系统）→ 整个生物体。 肌肉细胞：放弃了光合、感光等能力，专精收缩和舒张→化学能直接变机械运动→所有动物能"动"的物理基础。
埃迪亚卡拉动物群 · 寒武纪大爆发
▾
埃迪亚卡拉（~5.7亿年）：最早大型多细胞化石群。扁平叶状/圆盘状，没嘴没腿没眼——寒武纪前的"实验品"。 寒武纪大爆发（~5.4亿年）：几乎所有现代动物门类的祖先在几千万年内同时出现。为什么？①O₂终于够高 ②眼睛出现→捕食者能"看见"→猎物进化壳/刺/速度→军备竞赛启动 ③基因调控网络（Hox基因）成熟→小基因变化→大身体变化。
~3.7 亿年前
脊椎动物从海洋到陆地的进化
生命在海洋待了30+亿年。登陆要解决：重力（没浮力）、呼吸（鳃不行）、干燥（脱水）、繁殖（精卵不能空气中游）。
从鳃呼吸到肺呼吸
▾
鳃和肺干同一件事：O₂穿过薄膜入血，CO₂从血液排出。区别只是介质从水→空气。 某些鱼进化出"气囊"→水面吞空气→缺氧浅水中巨大优势→气囊逐步变肺。鱼鳍骨头变粗→支撑身体→变四肢。
两栖→爬行→羊膜卵
▾
两栖（类似蝾螈）能在陆地活动但必须回水里繁殖——卵没壳会干死。 后来进化出羊膜卵——壳+内部液体=胚胎自带小池塘→不需要回水边→深入内陆→爬行动物诞生（~3.1亿年前）。爬行动物从此统治陆地。
~2.3 亿–6600 万年前
恐龙的出现和哺乳动物的兴起
距今2.51亿年，95%海洋物种、70%陆地脊椎动物灭绝——史上最严重。可能原因：大规模火山→CO₂暴增→温室效应→海洋酸化→食物链崩溃。之后5000万年恢复，"生态真空"里恐龙崛起。
二叠纪-三叠纪灭绝事件
▾
距今2.51亿年，95%海洋物种、70%陆地脊椎动物灭绝——史上最严重。可能原因：大规模火山→CO₂暴增→温室效应→海洋酸化→食物链崩溃。之后5000万年恢复，"生态真空"里恐龙崛起。
恐龙——直立姿势的革命
▾
关键区别：腿的位置。其他爬行动物腿伸向两侧（展肢）。恐龙腿垂直在身体下方（直立/柱状）。 为什么这决定了谁称霸？效率（不需持续收缩肌肉防滑）、速度（推进效率高）、体型（展肢撑不住大体型，直立可以）。蜥脚龙30-100m、霸王龙12m、三角龙/剑龙/甲龙各种防御。统治1.65亿年（人类只有~30万年）。
哺乳动物——恐龙阴影下1.65亿年
▾
和恐龙几乎同时出现。但被压制：极小（鼠大小）、夜行、隐蔽。大型生态位被恐龙占完。在"缝隙"里悄悄进化出更复杂大脑、更敏锐嗅觉听觉。 距今6600万年，10km小行星撞地球→尘埃遮天→光合作用停→植物死→食草动物死→食肉动物死→食物链从底层崩塌。大型动物每天需大量食物→几周断粮就死→且没法躲进地洞。小型哺乳动物只需一点食物→靠种子昆虫苟活→钻进地洞躲过热浪→活下来→生态位全空→爆发。
~6600 万年前 → 现在
哺乳动物到人类的进化
~6500万年前。森林覆盖→树上生活需要：前向眼睛（立体视觉，跳跃必须判断距离）、灵活手指（抓握）、旋转手腕、更大大脑（视觉皮层+运动皮层）。你现在还保留这些——全是树上生活的遗产。
灵长类——树上生活塑造了我们
▾
~6500万年前。森林覆盖→树上生活需要：前向眼睛（立体视觉，跳跃必须判断距离）、灵活手指（抓握）、旋转手腕、更大大脑（视觉皮层+运动皮层）。你现在还保留这些——全是树上生活的遗产。
类人猿 → 直立行走
▾
类人猿（~2500万年）：没尾巴、臂行、更大脑、更复杂社会。人类也是类人猿。 直立行走（~700万年）：东非气候变干→森林退缩→被迫下树。直立优势：看更远、手腾出来、散热更好、长距离省力。南方古猿脑容量400-500cc，存在了约400万年。
工具与大脑——能人 → 直立人 → 智人
▾
能人（~300万年）：第一个造工具→切肉/敲骨髓→高能食物→支撑更大脑→更好工具→正反馈循环。 直立人（~200万年）：脑容量~900cc。两个关键突破——①用火→烹饪让食物更易消化→同样食物净能量增加→养更大的脑 ②走出非洲→到达亚洲和欧洲。 智人（~30万年）：关键创新=语言→知识脱离基因传递→积累速度从百万年级变成几代人。
人类和黑猩猩的DNA有98.8%是相同的
▾
1.2%差异→在几百万年自然选择中被放大→直立行走、语言、抽象思维、文明。微小的基因差异，巨大的表型差异。
